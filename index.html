About

I've been programming for a long time. I made my own video games as a kid, majored in CS and spent some time in grad school, and worked professionally for the last 10 years.

Several years ago I decided to focus on front-end web development. I'm a visual person. I love seeing the immediate results of my work and working to make things beautiful and easier to use. I also like the 'newness' of the field and how quickly things change with the latest innovations.

Over the years, I have had the opportunity to work with many different technologies in different roles at many different companies. Take a look at my resume to see the complete list. I also spend time developing my own side projects that I hope to one day turn into a profitable business.

Projects

Make It So

Github

Demo is currently offline

Make It So is a web-app disguised as a text adventure. The back-end is written in NodeJS with the Express framework and interacts with MongoDB. The front-end uses React, Backbone, and SASS.

Inspired by Star Trek: The Next Generation, you navigate the galaxy solving mysteries and resolving diplomatic crises. Use your crew and ship instruments to gather data then your wits and captain's instincts to make the right calls and bring peace to the galaxy.

Argie

Github
Dev Journal

Argie is my newest project. It is an online platform for interactive worlds.

The idea is that I spent a lot of time creating an interactive engine for MakeItSo, so why not add an easy-to-use interface to it and let people make their own online text adventures. My hope is to create a platform where online self-expression is easy and inspire others to create their own interactive fictional worlds.

The project is written in TypesScript with GraphQL, mongoose/MongoDB, and Jest unit testing with 100% code coverage. It is currently in gestation phase, but you can visit the public github and read my rambling developer journal.
