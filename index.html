Projects

Wrdie

https://wrdie.com

Closed Source. Github access is available on request

Wrdie is a platform for creating and playing online interactive stories and text adventures. It is 100% TypeScript using React, Node/Express, GraphQL, and MongoDB.

I was working on my own online text adventure (see below) when I realized that I am a much better programmer than I am a writter. So I decided to put an interface on the engine I had built and turn it into a platform that lets anyone create their own online interactive story.

4 years later and Wrdie is finally available in beta. The editor features a drag and drop tool that lets anyone create interactivity without having to write any code. My hope is to one day have a large community of creators and players all using Wrdie for their own interactive self-expression.

Make It So

http://makeitso.hiscores.net
Github

Make It So is a web-app disguised as a text adventure. The back-end is written in NodeJS with the Express framework and interacts with MongoDB. The front-end uses React, Backbone, and SASS.

Inspired by Star Trek: The Next Generation, you navigate the galaxy solving mysteries and resolving diplomatic crises. Use your crew and ship instruments to gather data then your wits and captain's instincts to make the right calls and bring peace to the galaxy.
